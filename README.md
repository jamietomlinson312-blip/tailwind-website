# Portfolio Website Practice

A small frontend practice project built to improve responsive web design, layout techniques, and modern UI development.

## Overview

This project is a multi-page portfolio website consisting of:

- Landing (Hero) page
- Gallery page
- Responsive navigation
- Mobile-friendly layout
- Modern, clean design

The goal of this project is to practice building a polished website, while following good project structure and version control practices.

---

## Features

- Responsive design
- Mobile navigation
- Hero section
- Image gallery
- Hover animations
- Clean component structure
- Accessible HTML semantics

---

## Built With

- HTML5
- Tailwind CSS
